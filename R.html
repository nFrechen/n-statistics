 , das Multitool der Datenausertung

R ist ein Software-Multitool für die Datenanalyse und -visualisierung. Es enthält unzählige Statistik-, Datenverarbeitungs- und Darstellungs-Funktionen, die an Universitäten und von anderen Fachleuten auf freiwilliger Basis entwickelt wurden. Mehr und mehr Datenanalysten steigen von kommerziellen Tools wie SPSS, SAS, Matlab, Maple, Mathematica, oder Excel auf R um und tragen zu dessen Weiterentwicklung bei.

  umfasst über 9000 Pakete

Die Stärke von R liegt in seiner Vielfalt. Über 9000 Pakete stehen zur Verfügung für alle erdenklichen Anwendungsfälle. Und stetig werden es mehr, da unzählige Entwickler nach ihren Bedürfnissen Anwendungsfälle ergänzen.

  ist Open-Source Software

das bedeutet:

frei nutzbar

  • Es fallen keine Lizenzgebühren zur Nutzung der Software an
  • Nutzen Sie die Software auf beliebig vielen Rechnern
  • Geben Sie die Software an Ihre Clienten/Kunden weiter
  • Es besteht keine Gefahr, dass die Software durch Probleme mit der Lizenzierung zeitweise oder dauerhaft nicht mehr genutzt werden kann
  • Sie müssen sich nicht entscheiden "auf welches Pferd Sie setzen". Da keine Zusatzkosten entstehen können Sie R problemlos parallel mit anderer Software nutzen.

frei nachvollziehbar

  • Open-Source bedeutet, dass Sie den gesamten Quelltext einsehen können. Nicht nur den, den wir für Sie geschrieben haben, sondern auch den aller verwendeten Funktionen und Pakete.
  • Im Wissenschaftlichen Kontext garantiert die Nachvollziehbarkeit des Programmtextes die Reproduzierbarkeit der Forschungsergebnisse und fördert damit die Akzeptanz und Verbreitung Ihrer Ergebnisse.

frei veränderbar

  • Sie können frei nach Belieben Änderungen vornehmen
  • Passen Sie die Auswertung an neue bzw. andere Datensätze an
  • Verwenden Sie die gleiche Analyse in leicht verändertem Kontext

frei erweiterbar

  • Erweitern Sie die von uns angefertigte Analyse um weitere Bestandteile
  • Entwickeln Sie die Analyse selbstständig weiter
  • Nutzen Sie die Analyse als Bestandteil von größeren Meta-Analysen

eine offene und hilfsbereite Community

  • Finden Sie hilfsbereite Entwickler auf Stackoverflow.com, die Ihnen bei Ihren Problemen helfen.
  • Regen Sie Programmerweiterungen und Zusatzfunktionen an und staunen Sie wie schnell die Community Ihre Anregungen aufgreift!
  • Durch gegenseitigen Austausch erreichen alle mehr

Was muss ich bei der Nutzung von Open-Source Software beachten?

Die Software untersteht der GPL-Lizenz. Diese legt fest, dass jede Software, die damit entwickelt wird wiederum der selben Lizenz unterstehen muss. Das bedeutet für Sie, wenn Sie die Software weiter geben, können Sie dies nicht unter einer anderen (z.B. kommerziellen) Lizenz machen. Ausserdem sind Sie verpflichtet, den Quellcode offen zu legen, sobald Sie die Software weiter geben. Solange die Software jedoch in Ihrem Unternehmer bleibt, können Sie damit nach Ihren eigenen Vorgaben umgehen und niemand kann den Quellcode einklagen. Die Analyseergebnisse, die die Software ausspuckt, unterstehen nicht der GPL-Lizenz. Diese können Sie also bedenkenlos nach Ihrer eigenen Lizenz verbreiten.

Lernen Sie  

Auf den Seiten von codeschool.com können Sie R ausprobieren, ohne es installieren zu müssen.

Laden Sie R von r-project.org herunter um es zu installieren. Am besten installieren Sie sich auch direkt den eigens für R entwickelten Editor RStudio.

Wenn Sie R bereits installiert haben, können Sie mit dem Paket swirl interaktive Tutorials direkt in der Konsole von R absolvieren. In unserem Blog finden Sie eine Anleitung für die Installation von swirl-Tutorials auf Deutsch.

Oder sprechen Sie uns einfach an und wir geben Ihnen eine persönliche Einführung und zeigen Ihnen, wie Sie R für Ihre Anwendungsfälle einsetzen können.
